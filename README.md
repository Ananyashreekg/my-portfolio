# Ananyashree K G - Portfolio

A high-performance, dark-themed portfolio built with **React** and **Tailwind CSS**.

## 🚀 Overview

This portfolio showcases my expertise as a **Backend & Data Engineer**, highlighting my work with Node.js, ClickHouse, PostgreSQL, and Python. It features a modern, premium design with smooth transitions and a responsive layout.

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Styling:** Custom CSS for glassmorphism and smooth animations
- **Backend Knowledge (Showcased):** Node.js, ClickHouse, Redis, PostgreSQL, Python, AWS S3
- **Tools:** Git, ELK Stack, Power BI

## 📂 Project Structure

- `src/components/`: Reusable React components for each section (About, Skills, Projects, etc.)
- `src/App.js`: Main application entry point managing sections and navigation.
- `src/index.css`: Global styles and custom animation tokens.

## 🏃 Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/Ananyashreekg/my-portfolio.git
   ```
2. Install dependencies
   ```bash
   npm install
   ```
3. Start the development server
   ```bash
   npm start
   ```
